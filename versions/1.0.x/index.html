Simply Powerful

Robust UI tools at your fingertips.

AlloyUI is a framework built on top of YUI3 that provides a simple API for building high scalable applications.

get started

it’s really easy!

1 - Copy and paste

First load the seed file, if you haven't yet.

<script src="https://cdn.alloyui.com/1.0.3/aui/aui-min.js"></script>

2 - Start using Alloy UI

Then initialize AlloyUI and load a module, e.g., AutoComplete.

AUI().use(
  'aui-autocomplete',
  function(A) {
    // code goes here
  }
);
