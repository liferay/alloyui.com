Image Viewer

The Image Viewer component allows users to view and navigate through a collection of images.

Getting Started

First load the seed file if you haven't yet. 

<script src="http://deploy.alloyui.com/build/aui/aui.js"></script>

Then initialize AlloyUI and load the Image Viewer module. 

AUI().use('aui-image-viewer', function(A) {
  // code goes here
});

Using Image Viewer

Image Viewer uses images that are linked in the HTML: 

<div id="gallery">
  <!-- Image being displayed with Image Viewer -->
  <a href="image-1.jpg">
    <!-- Thumbnail of image to be displayed -->
    <img src="image-1-thumbnail.jpg" />
  </a>
  <a href="image-2.jpg">
    <img src="image-2-thumbnail.jpg" />
  </a>
  <a href="image-3.jpg">
    <img src="image-3-thumbnail.jpg" />
  </a>
</div>

Creat a new instance of Image Viewer referecing the linked images in the HTML, and render it! 

AUI().use('aui-image-viewer', function(A) {

  var imageViewer = new A.ImageViewer({
    links: '#gallery a'
  }).render();

});

Configuring Image Viewer

Parameters can be passed into your instance of Image Viewer to change attibutes such as image captions, image positioning, and when images within the viewer are loaded. 

AUI().use('aui-image-viewer', function(A) {

  var imageViewer = new A.ImageViewer({
    links: '#gallery a',
    //captionFromTitle uses the DOM title attibute as image caption
    captionFromTitle: true,
    preloadAllImages: true,
    //preloadNeighborImages: true,
    centered: false
  }).render();

});

Further configuration includes declaring what elements are to be used to house the caption, navigation arrows, close button, and even the loader, which houses the image being displayed.

Note: when attributes such as loader, arrowLeftEl, etc., are not specified, the HTML element is generated by Image Viewer. 
AUI().use('aui-image-viewer', function(A) {

  var imageViewer = new A.ImageViewer({
    links: '#gallery a',
    loader: '#loader',
    arrowLeftEl: '#arrowLeft',
    arrowRightEl: '#arrowRight',
    captionEl: '#caption',
    closeEl: '#close'
  }).render();

});

For more information about configuration, check out our API Docs.
